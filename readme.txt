Данный проект реализован с помощью паттерна MVC, что позволит легче его масштабировать и
    поддерживать.

Далее приведены основные функции и описание методов и файлов проекта:

    Папка Config содержит конфигурационные файлы и методы, такие как:
        Данные подключения к БД, и их дальнейшая обработка.
        Основные константы проекта
        Функции (в данном случае только функция редиректа)

    Папка Core - ядро приложения, хранит в себе основные и базовые классы:
        Traits - ConnectionTrait - отвечает только за метод connect для подключение к БД с помощью PDO
                 QueryTrait - основные методы для работы с моделями такие как сохранение, обновление, пагинация и.т.д
        Контроллер - базовый контроллер для наследования
        FileUploadService - класс для загрузки и удаления файлов с сервера
        Model - базовый абстрактный класс
        Router - служит для удобства определения маршрутов в приложении,
                метод Add выполняет преобразования маршрута в регулярное выражение
                dispatch - выполняет основную функцию, сопоставляет маршрут с запрошенным URL с помощью метода match
                            который в свою очередь проверяет с помощью ранее созданного регулярного выражения эти совпадения.
                            в случае успеха сверяется метод маршрута и запроса, создается экзепляр указанного контроллера
                            и вызывает указанный метод контроллера

        View - отбражение представлений с возможностью передать параметры в виде переменных
