Данный проект реализован с помощью паттерна MVC, что польволит легче его масштабировать и
    поддерживать.

Далее приведены основные функции и описание методов и файлов проекта:

    Папка Config служит для создание подключения к БД, файл configs содержит данные для подключения.
    Папка Core - ядро приложения, хранит в себе основные и базовые классы
    Класс Model отвечает за подключение с помощью PDO
    Класс Router отвечает за маршрутизацию веб-приложения.
        Он обрабатывает входящий URL, сопоставляет его с маршрутами указанными в файле web.php и вызывает
        соответствующие методы контроллеров.
